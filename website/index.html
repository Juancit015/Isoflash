USB Management

Automatic detection of USB devices with real-time scanning every 2 seconds. Instantly identifies drives with Ventoy installed and displays partition info.

Ventoy Install / Update

One-click Ventoy installation or update on any USB drive. Auto-downloads Ventoy if not found locally. Supports both MBR and GPT layouts.

Built-in Distro Catalog

Browse 70+ Linux distributions and Windows ISOs. Filter by category — Rolling Release, LTS, Server, Security, Gaming, Windows — and find the right ISO in seconds.

Download Manager

Queue multiple ISOs with start/pause/resume control. Configurable speed limits (Low/Medium/High/Max) and progress tracking for every download.

ISO Flashing

Copy ISO files directly to Ventoy-prepared USB partitions with a single click. Automatic free-space verification and buffered I/O for reliable transfers.

Operation Logs

Detailed per-operation logging with timestamps. Expandable log viewer lets you track every action, diagnose errors, and review history.

How to use Isoflash

Get from zero to a bootable USB in four steps

1

Plug in your USB drive

Isoflash automatically detects connected USB devices. The Dashboard panel shows all drives with details like size, model, filesystem, and whether Ventoy is already installed.

2

Install Ventoy

Select your USB drive from the list and click Install Ventoy. Isoflash downloads Ventoy automatically if needed, then installs it (requires root via sudo or pkexec).

3

Download an ISO

Open the Catalog tab, browse or search 70+ distributions, filter by category, and click Add to downloads. Manage your download queue with pause/resume and speed controls.

4

Flash to USB

Go to the Flash tab, select your Ventoy-prepared USB and the downloaded ISO, then click Flash to Ventoy. The ISO is copied to the USB and ready to boot.

Frequently Asked Questions

Common questions about Isoflash

Isoflash is a desktop application written in Rust that simplifies creating bootable USB drives for Linux and Windows. It combines automatic USB detection, Ventoy installation, a built-in catalog of 70+ distributions, an ISO download manager, and flashing tools — all in one modern GUI built with egui.

Currently Isoflash only supports Linux. Windows and macOS support is planned for a future release. The tool relies on Linux-specific utilities like lsblk, udisksctl, and sudo/pkexec.
